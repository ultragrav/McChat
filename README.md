# McChat
A lightweight Minecraft chat wrapper

## Installation
### Maven:
```xml
<repository>
    <id>UltraGrav</id>
    <url>https://mvn.ultragrav.net</url>
</repository>

<dependency>
    <groupId>net.ultragrav</groupId>
    <artifactId>McCHat</artifactId>
    <version>1.0.0</version>
</dependency>
```
